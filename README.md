# Social Media API
 
  ## Description
  A REST API for a social media app. Built with Express, Mongoose, and MongoDB.

  ## Installation
  Make sure you have mongoDB installed.
  Clone the Repo
  Add the dependencies ("npm i")
  "npm start" to start the server on your localhost.
